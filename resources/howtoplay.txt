SIX LETTER WORDLE

Wordle is a logic word game invented in 2021 by a Welsh software engineer, Josh
Wardle.  The game is now owned by the New York Times.

SIX LETTER WORDLE was written by Joe Bentley in 2023. Joe wrote it as a variation
of the original game.  The New York Times Wordle is a web-based application.  
Joe's game was written in C++ using SFML libraries for graphics, sound, mouse and 
keyborad controls.  It currently runs on Windows, Mac and Linux.
